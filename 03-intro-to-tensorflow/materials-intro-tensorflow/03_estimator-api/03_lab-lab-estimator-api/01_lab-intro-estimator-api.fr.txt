Dans cet atelier, nous allons
écrire notre premier modèle à l'aide de l'API Estimator. Vous allez apprendre à définir
des caractéristiques d'entrée pour le modèle
à l'aide de colonnes de caractéristiques, à choisir un estimateur prédéfini
et à l'entraîner sur un petit ensemble de données de démonstration
qui rentre en mémoire. Suivez l'atelier seul en lisant
les commentaires du bloc-notes et en exécutant les cellules de code. Je vous encourage à modifier le code
pour vérifier que vous avez bien compris comment fonctionne chaque partie,
puis à revenir à la vidéo récapitulative, dans laquelle nous passerons
tout cela en revue. Notez que vous avez trois tentatives
pour terminer l'atelier, donc ne vous inquiétez si vous manquez
de temps lors de votre premier essai.
