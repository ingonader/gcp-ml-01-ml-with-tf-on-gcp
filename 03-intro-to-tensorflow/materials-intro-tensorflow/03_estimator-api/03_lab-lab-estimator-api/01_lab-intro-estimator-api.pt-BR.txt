Neste laboratório, vamos escrever nosso
primeiro modelo usando a API Estimator. Você verá como definir recursos de entrada
para o modelo usando colunas de recursos, como escolher um estimador pré-fabricado e como treiná-lo em um conjunto de dados
de demonstração que cabe na memória. Pratique com o laboratório, lendo os
comentários do bloco de notas, e executando as células do código. Edite o código para garantir
que você entenda o que cada parte faz e, em seguida, volte para o vídeo de
finalização para analisarmos juntos. Você tem três tentativas
para concluir o laboratório. Então, não se preocupe se você ficar
sem tempo na primeira.
