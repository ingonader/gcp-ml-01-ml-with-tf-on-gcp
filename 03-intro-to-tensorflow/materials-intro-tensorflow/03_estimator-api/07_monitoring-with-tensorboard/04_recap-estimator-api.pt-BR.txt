Com isso, concluímos nossa tour
pela API Estimator. Ela permite que você crie modelos
que vão desde pequenos protótipos até grandes modelos
prontos para produção. Com o conjunto avançado
de estimadores pré-fabricados, você pode experimentar
modelos padrão com rapidez. E você também pode criar
seu próprio estimador personalizado. Vamos abordar isso mais tarde neste curso. Então, quando você está ficando sério, a API deixa você conectar
conjuntos de dados sem memória ao treinamento e avaliar e colocar funções
com a API Dataset. Treinar e avaliar inicia um loop de treino
que alterna o treinamento e o teste, para que você possa monitorar
o andamento no TensorBoard. Ele também implementa
treinamento distribuído e, por fim, os exportadores permitem adicionar
o código de cola necessário na produção e implantar o modelo atrás de uma API totalmente gerenciada
com escalonamento automático. Isso não é tão complicado assim, são apenas centenas de linhas de código
boilerplate que o TensorFlow fornece, encapsuladas na API Estimator, para que você possa se concentrar
em seus dados e modelos.
