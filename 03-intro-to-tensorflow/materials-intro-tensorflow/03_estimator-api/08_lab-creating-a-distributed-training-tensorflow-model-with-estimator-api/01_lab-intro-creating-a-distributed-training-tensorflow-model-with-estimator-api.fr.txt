Dans cet atelier, vous allez voir
pourquoi il est important de consulter vos métriques de validation
lors de l'entraînement. Pour cela, je vais vous présenter
la fonction "train_and_evaluate", ainsi qu'un nouvel outil
de visualisation, TensorBoard. Au passage, "train_and_evaluate"
implémente aussi l'entraînement distribué. C'est inclus dans la solution. Nous allons apprendre
à utiliser cette fonction et à surveiller l'entraînement
à l'aide de TensorBoard. Pratiquez l'atelier seul en lisant
les commandes du bloc-notes et en exécutant les cellules de code. Je vous invite à modifier le code pour vérifier que vous avez bien compris
ce que fait chaque partie, puis à revenir à la vidéo récapitulative, dans laquelle
nous passerons tout cela en revue. Notez que vous avez trois essais
pour terminer l'atelier, donc ne vous inquiétez pas si vous manquez
de temps lors de votre premier essai.
