Neste laboratório, você aprenderá
a importância de observar a matriz de validação enquanto
o treinamento está em andamento. Para isso, apresentaremos a função
do estimador train_and_evaluate, bem como uma nova
ferramenta de visualização, o TensorBoard. E a propósito, train_and_evaluate
também implementa treinamento distribuído. Estamos recebendo isso de graça. Especificamente, abordaremos como usar train_and_evaluate e como monitorar
o treinamento usando o TensorBoard. Pratique com o laboratório lendo os comandos do bloco de notas e
executando as células do código. Incentivo você a editar o código para
entender o que cada parte faz e, depois, volte ao vídeo de finalização,
quando faremos isso juntos. Você terá três tentativas
para concluir o laboratório, então não se preocupe se ficar
sem tempo na primeira tentativa.
