In diesem Lab trainieren Sie Datasets,
die zu groß für den Arbeitsspeicher sind. TensorFlow hat eine API namens
Datasets, die damit umgehen kann und Ihrem Modell Daten zuführt, während
es sie progressiv von der Festplatte lädt. Üben Sie selbst mit dem Lab, indem Sie
die Anmerkungen des Notebooks lesen und die Codezellen ausführen. Bearbeiten Sie ruhig den Code, um
jeden Teil davon richtig zu verstehen, und kehren Sie dann zum Video
mit der Nachbesprechung zurück, wo wir ihn gemeinsam durchgehen. Beachten Sie, dass Sie drei
Versuche haben, das Lab abzuschließen. Also, machen Sie sich keine Sorgen, wenn
Ihnen im ersten Versuch die Zeit ausgeht.
