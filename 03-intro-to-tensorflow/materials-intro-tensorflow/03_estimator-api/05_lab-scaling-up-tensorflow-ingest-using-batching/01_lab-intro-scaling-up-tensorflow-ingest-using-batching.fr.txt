Dans cet atelier, vous allez
apprendre à effectuer un entraînement sur des ensembles de données volumineux
qui ne rentrent pas en mémoire. L'API Dataset de TensorFlow peut
gérer cela et alimenter votre modèle tout en chargeant les données à partir
d'un disque de manière progressive. Pratiquez cet atelier seul en lisant
les commentaires du bloc-notes et en exécutant les cellules de code. Je vous invite à modifier le code
pour vous assurer que vous comprenez bien chaque partie. Puis revenez à la vidéo récapitulative, dans laquelle nous passerons
tout cela en revue. Vous disposez de trois essais
pour réaliser l'atelier. Ne vous inquiétez donc pas
si vous manquez de temps la première fois.
