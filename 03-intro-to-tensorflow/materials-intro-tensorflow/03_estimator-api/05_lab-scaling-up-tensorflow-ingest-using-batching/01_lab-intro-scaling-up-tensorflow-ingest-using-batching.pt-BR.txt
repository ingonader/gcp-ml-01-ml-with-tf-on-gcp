Neste laboratório, você vai aprender a
treinar grandes conjuntos de dados que não cabem na memória. O TensorFlow tem uma API chamada
Datasets que pode fazer isso e alimentar seu modelo enquanto carrega
os dados do disco de maneira progressiva. Pratique com o laboratório lendo os
comentários do bloco de notas e executando as células do código. Edite o código para
entender o que cada parte faz. Em seguida, volte para o vídeo final,
quando veremos juntos. Você tem três tentativas
para concluir o laboratório. Então não se preocupe se você ficar
sem tempo na primeira.
