Bonjour, je suis Valiapa Lakshmanan, mais on m'appelle Lak. L'outil qui va nous servir
à écrire des programmes de ML s'appelle TensorFlow. Dans ce cours, mes co-auteurs et moi-même allons vous présenter TensorFlow. Il s'agit du troisième cours de spécialisation
en machine learning sur GCP. Dans le premier cours, vous avez appris à traduire des problématiques métier
en problèmes de machine learning. Dans le second, vous avez abordé
l'aspect pratique du machine learning et appris à créer
des ensembles de données utilisables dans vos opérations
de machine learning. Maintenant que les données sont en place, vous pouvez développer
des programmes de machine learning. Dans ce cours, vous allez apprendre
à créer des modèles de machine learning dans TensorFlow. Nous commencerons par parler
de Core TensorFlow. Il s'agit de TensorFlow en tant que bibliothèque
de programmation numérique. Vous allez apprendre à utiliser
les bibliothèques TensorFlow pour résoudre des problèmes numériques. Lorsqu'on écrit des programmes, il est judicieux de connaître
les erreurs les plus fréquentes ainsi que la façon de les réparer. Vous allez donc apprendre
à réparer et déboguer les "pièges" TensorFlow les plus courants. Nous nous intéresserons ensuite
à l'API Estimator qui offre le plus haut niveau d'abstraction
dans TensorFlow pour l'entraînement, l'évaluation et la distribution
des modèles ML. Vous apprendrez à utiliser
tf_estimator pour créer, entraîner et évaluer
un modèle de machine learning. Enfin, vous apprendrez à exécuter
les modèles TensorFlow que vous créez sur Cloud ML Engine. C'est l'infrastructure gérée par Google sur laquelle vous exécuterez
vos modèles TensorFlow. Vous découvrirez comment entraîner,
déployer et produire vos modèles de machine learning 
à grande échelle avec Cloud Machine Learning Engine. Bienvenue dans TensorFlow.
