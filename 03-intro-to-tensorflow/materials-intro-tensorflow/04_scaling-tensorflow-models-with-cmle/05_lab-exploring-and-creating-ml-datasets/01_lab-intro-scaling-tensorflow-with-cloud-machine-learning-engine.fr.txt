Dans cet atelier, nous allons utiliser
un modèle TF déjà développé pour prédire des tarifs de courses de taxi
et nous allons l'empaqueter pour pouvoir l'exécuter sur CMLE. Pour l'instant, nous allons l'exécuter
sur un petit ensemble de données. Le modèle développé est plutôt simpliste
et sa précision n'est donc pas optimale. Toutefois, cet atelier présente les bases
de l'exécution de modèles TF sur Cloud ML. Nous allons notamment voir comment
empaqueter un modèle TF, exécuter un entraînement local
et sur le cloud, déployer le modèle sur le cloud et l'appeler
pour effectuer des prédictions. Suivez cet atelier seul en lisant
les commentaires du bloc-notes et en exécutant les cellules de code. Je vous encourage à modifier le code
pour vous assurer que vous avez bien compris
chaque partie. Puis, revenez à la vidéo récapitulative,
dans laquelle nous en parlerons en détail.
