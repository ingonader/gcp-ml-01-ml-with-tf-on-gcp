En este lab, usaremos un modelo de TensorFlow
desarrollado previamente para predecir tarifas de taxi y lo empaquetaremos
para que se ejecute en CMLE. Por ahora, lo ejecutaremos
con un conjunto de datos pequeño. El modelo desarrollado es sencillo y, por lo tanto,
no es realmente muy preciso. Lo importante es que este lab
mostrará los aspectos básicos de ejecutar modelos
de TensorFlow en Cloud ML. Específicamente, cubriremos cómo
empaquetar un modelo de TensorFlow ejecutar el entrenamiento
de forma local o en la nube implementar el modelo en la nube
e invocarlo para realizar predicciones. Para practicar el lab,
lea los comentarios del notebook y ejecute las celdas de código. Lo invito a editar el código
para entender lo que hace cada parte. Luego, mire el video final
para revisar todo juntos.
