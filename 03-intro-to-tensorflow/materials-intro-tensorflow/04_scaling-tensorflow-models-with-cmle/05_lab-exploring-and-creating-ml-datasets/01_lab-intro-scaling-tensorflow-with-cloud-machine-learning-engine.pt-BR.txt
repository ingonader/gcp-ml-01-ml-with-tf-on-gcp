Neste laboratório, usaremos um modelo TensorFlow criado
anteriormente para prever tarifas de táxi e o empacotaremos
para ser executado no Cloud ML Engine. Por enquanto, executaremos isso
em um pequeno conjunto de dados. O modelo desenvolvido
é bastante simplista e, portanto,
a precisão do modelo não é grande. No entanto, este laboratório ilustrará o básico da execução
dos modelos do TensorFlow no Cloud ML. Especificamente, falaremos sobre como
empacotar um modelo do TensorFlow, executar o treinamento
localmente e no Cloud, implantar o modelo no Cloud e
invocar o modelo para realizar previsões. Pratique o laboratório sozinho
lendo os comentários do bloco de notas e executando as células do código. Recomendo editar o código para que
você entenda o que cada parte faz. E depois volte para o vídeo final,
onde veremos tudo juntos.
