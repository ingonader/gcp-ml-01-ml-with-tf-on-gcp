Neste laboratório, você
verá o impacto de criar conjuntos de dados de aprendizado
de máquina de jeitos diferentes. A repetibilidade é importante
no aprendizado de máquina. Imagine que você tenha
feito alterações no seu modelo, como nos parâmetros
e hiperparâmetros. E os dados subjacentes nos
treinamentos seguintes também mudam. Você não saberá se
o desempenho do seu modelo melhorou, com base no desempenho real
dos modelos que estão se saindo melhor, ou se o instantâneo de dados
usado naquele treinamento estava mais fácil porque os dados mudaram. Se mantiver os dados constantes
à medida que altera o modelo, você poderá ajustar seu modelo e executá-lo de novo no mesmo 
conjunto de dados de experimentação. Neste laboratório, você
praticará como criar, dividir e manter esses conjuntos
de dados constantes. Experimente. As soluções de todos esses laboratórios
estão no repositório de códigos no GitHub. Tudo é de código aberto. E você tem acesso ao repositório
mesmo depois do fim do curso. Nós incentivamos você. Como gosto de dizer: "bons artistas
copiam, grandes artistas roubam". Use esse código como ponto de partida
para qualquer projeto futuro de ML. Experimente o laboratório e
volte para a solução.
