Wie Sie in meiner kurzen 
Demo sehen konnten, war ich in der Lage, große Datasets 
sofort und interaktiv abzufragen. Es handelte sich dabei mit SQL:2011
um eine Standardabfrage. BigQuery bietet viele Funktionen,
um Daten hochzuladen. Sie können zum Beispiel 
Dateien in das GUI hochladen, in Cloud Storage speichern und darauf verweisen oder zu BigQuery streamen. Durch die große Auswahl an APIs können
Sie Dateien aus BigQuery einfach exportieren. Nach Ausführung einer SQL-Abfrage lässt sich
das Ergebnis im gewünschten Format speichern. Der Speicherplatz ist auch recht günstig. Zum Schluss kommt 
das für uns Allerwichtigste: Datalab und BigQuery sind 
miteinander kompatibel. Somit können Sie Daten abfragen und ansehen,
in einen Pandas-DataFrame exportieren und mithilfe von Python plotten. Und genau das werden Sie jetzt tun.
