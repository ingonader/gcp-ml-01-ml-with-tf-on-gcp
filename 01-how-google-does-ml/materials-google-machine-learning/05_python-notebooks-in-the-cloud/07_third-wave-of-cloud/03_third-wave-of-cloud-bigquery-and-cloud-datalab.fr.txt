Dans ma démonstration, j'ai pu exécuter une requête
de façon interactive et ponctuelle. J'ai utilisé SQL:2011,
un langage standard. L'introduction de données
dans BigQuery est très flexible. Vous pouvez notamment
importer des fichiers via l'interface Web,
les entreposer dans Cloud Storage pour les sélectionner,
ou diffuser des données dans BigQuery. L'exportation des données depuis BigQuery
est facilitée grâce aux diverses API. Exécutez une requête SQL et enregistrez
le résultat au format de votre choix. Le stockage est également peu onéreux. Enfin, et c'est un point clé
pour nos besoins, Datalab s'intègre parfaitement à BigQuery. Nous pouvons ainsi explorer des données,
exécuter une requête, exporter les résultats dans un "dataframe"
Pandas et tracer un graphique dans Python. C'est précisément
ce que nous allons faire.
