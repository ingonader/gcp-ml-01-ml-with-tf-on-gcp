En mi demostración rápida,
vieron que pude consultar grandes conjuntos de datos
de manera interactiva y ad hoc. Hice la consulta con SQL 2011,
un estándar. Colocar los datos en BigQuery
es una tarea muy flexible. Pueden hacerlo
mediante la carga de archivos desde la GUI, colocándolos
en Cloud Storage y redirigiéndolos, o incluso
transmitiéndolos hacia BigQuery. Pueden exportar desde BigQuery
fácilmente gracias a la variedad de API. Pueden ejecutar una consulta de SQL
y guardar los resultados en el formato que deseen. El almacenamiento es bastante económico. Y, finalmente,
clave para nuestros propósitos Datalab se integra fácilmente con BigQuery de modo que podemos explorar datos,
ejecutar una consulta exportar a un DataFrame de Pandas
y graficar con Python. De hecho, eso es exactamente
lo que harán a continuación.
