ad-hoc em conjuntos de dados grandes de modo interativo. Minha consulta em si foi

o upload de arquivos da GUI da Web para testá-los

devido às variedades das APIs. Execute uma consulta SQL e salve o
resultado no formato que quiser. O armazenamento também é
muito barato. E, por fim, algo essencial
para os nossos propósitos, o Datalab se integra muito bem
com o BigQuery, então podemos explorar dados,
executar uma consulta, exportar para um dataframe do Pandas e
