Neste laboratório, você clonará o repositório do GitHub do curso
na instância do Datalab e salvará tudo no seu projeto. Depois, usará um bloco de notas para
testar APIs de aprendizado de máquina. Como antes, os laboratórios fazem com que
você passe por um exemplo prático. Como não podemos ajudá-lo pessoalmente
a depurar o código, optamos por trabalhar desta forma. Para que você entenda o que
está sendo feito, altere o código e veja se consegue entender o
impacto dessas alterações. Por fim, remova o código de uma
seção crítica do bloco de notas. As seções críticas costumam ficar nas
seções de desafio dos laboratórios. Veja se você consegue escrever o
código sozinho. E se você não conseguir, consulte a solução.
