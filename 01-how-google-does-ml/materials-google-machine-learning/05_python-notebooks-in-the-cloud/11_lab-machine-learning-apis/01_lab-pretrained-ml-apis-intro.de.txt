In diesem Lab klonen Sie
das GitHub-Repository des Kurses in Ihrer Datalab-Instanz
und speichern alles in Ihrem Projekt. Danach können Sie mit einem Notebook
die Machine Learning APIs testen. Wie zuvor geht es bei den Labs darum,
ein Beispiel durchzuspielen. Weil wir Ihnen nicht dabei helfen können, Ihren Code zu debuggen,
haben wir diesen Weg gewählt. Damit Sie nachvollziehen können,
was passiert, können Sie versuchen, den Code zu ändern und zu beobachten,
was dadurch passiert. Entfernen Sie dann den Code aus einem wichtigen Abschnitt des Notebooks. Diese Abschnitte werden oft im Bereich "Herausforderung"
in den Labs hervorgehoben. Testen Sie, ob Sie 
den Code selbst schreiben können. Sollten Sie nicht weiterkommen,
können Sie in der Lösung nachsehen.
