En este lab, clonarán el repositorio
de GitHub del curso en su instancia de Datalab
y guardarán todo en su proyecto. Luego, trabajarán con el cuaderno para
probar las API de aprendizaje automático. Igual que antes, los labs a menudo
son una explicación del ejercicio. Ya que no podemos ayudarlos
personalmente a depurar su código lo hacemos de esta manera. Para asegurarse de comprender
lo que se está haciendo intenten cambiar el código
y vean si entienden el impacto de esos cambios. Finalmente, quiten el código
de una sección esencial del cuaderno. Las secciones esenciales a menudo
se señalan en la parte de desafío de los labs y vean si pueden
escribir el código ustedes mismos. Si no pueden avanzar,
vuelvan a ver la solución.
