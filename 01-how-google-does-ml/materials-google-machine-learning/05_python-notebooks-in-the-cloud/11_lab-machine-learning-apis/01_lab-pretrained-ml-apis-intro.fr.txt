Dans cet atelier, vous allez cloner le dépôt GitHub de ce cours
dans votre instance Datalab, et l'enregistrer dans votre projet. Vous pourrez alors essayer
les API de machine learning à l'aide d'un bloc-notes. Les ateliers vous permettent
de passer en revue un exemple concret. Comme nous ne pouvons pas
vous aider à déboguer votre code, nous avons choisi de procéder ainsi. Pour vous assurer que vous comprenez bien
les conséquences d'une modification, essayez de changer le code pour comprendre
l'impact de vos changements. Enfin, supprimez le code
d'une section stratégique du bloc-notes. Ces sections sont souvent mentionnées
dans les défis proposés dans les ateliers. Essayez d'écrire vous-même le code. Si vous n'y parvenez pas, référez-vous à la solution.
