Nun ein Blick auf Datalab. Datalab ist ein Jupyter-Notebook,
auf dem Sie Python, Bash usw. ausführen können. Angenommen ich bin in dieser
Zelle, einer Codezelle. Wenn ich hier nun den Python-Code "x=5, y=x+5, print y" einfüge und
die Eingabetaste drücke, erhalte ich die Ausgabe. Ein so einfacher Vorgang macht
die Leistung eines Notebooks aus. Die Ausgabe des Programms
ist in den Code selbst eingebettet. Sie können auch Markdowns hinzufügen. Hier gebe ich beispielsweise
einen Titel und etwas Text ein. Bei Ausführung erhalte
ich dann ein Markdown. Sie haben also alles
im gleichen Notebook: Code, Ausgaben und Kommentare. Wenn Sie ein Notebook ausführen, können Sie alle Zellen auf einmal löschen. Die gesamte Ausgabe wird damit gelöscht. Sie können auch alle
Zellen gleichzeitig ausführen. Daraufhin werden alle Zellen ausgeführt. Dies ist die grundlegende
Arbeitsweise in Datalab-Notebooks.
