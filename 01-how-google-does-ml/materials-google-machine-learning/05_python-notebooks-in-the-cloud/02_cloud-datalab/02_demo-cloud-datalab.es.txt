Así se ve Datalab,
que es un cuaderno de Jupyter un lugar donde pueden
ejecutar Python, Bash, etcétera. Por ejemplo,
en esta celda de código escribiré código de Python. x=5, y=x+5
print y Si lo ejecuto, obtengo la salida. El poder de un cuaderno
está en algo tan simple como esto. La salida del programa
está integrada en el código mismo. También, puedo agregar formato de marcado. Puedo indicar, por ejemplo,
"Aquí hay un título" y un poco de texto. Y cuando lo ejecuto,
obtengo un formato de marcado. La idea es que tienen código,
salidas, comentarios todo en el mismo cuaderno. Algo que harán a menudo
cuando ejecuten cuadernos es seleccionar "Clear all Cells"
y todas las salidas desaparecen. Luego, pueden seleccionar "Run all Cells"
y todas las celdas se ejecutan. Este es el proceso común cuando
se trabaja con el cuaderno de Datalab.
