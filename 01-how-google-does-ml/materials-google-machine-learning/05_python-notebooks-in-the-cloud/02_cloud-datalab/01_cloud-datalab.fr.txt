Utilisez-vous les blocs-notes
IPython ou Jupyter de nos jours ? Les data scientists optent de plus en plus
pour des blocs-notes autodescriptifs, partageables, exécutables pour analyser des données
ou faire du machine learning. Datalab est basé sur Jupyter
et il est Open Source. Voici à quoi ressemble
l'interface de Datalab. Notez les sections de code
où balisage et résultat sont intercalés. C'est cette intercalation
qui rend ce type de calcul si utile. Ces blocs-notes servent à l'analyse
de données et au machine learning. Vous pouvez exécuter le code en cliquant sur le bouton "Run" ou en appuyant
sur les touches Maj et Entrée. Notez que le résultat inclut non seulement
une ligne de commande, mais aussi des graphiques. La section en rouge contient le balisage
qui permet de justifier votre approche. L'une des sections en vert contient un bouton pour exporter le bloc-notes
comme fichier autonome. L'autre section en vert
vous permet de valider vos modifications dans un dépôt
de code sur Google Cloud Platform. Dans cet atelier, nous allons voir
comment faire une extraction et un commit dans un dépôt de code.
