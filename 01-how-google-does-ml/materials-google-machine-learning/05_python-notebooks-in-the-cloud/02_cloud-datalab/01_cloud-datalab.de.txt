Verwenden Sie IPython 
oder Jupyter-Notebooks? Data Scientists nutzen 
heutzutage immer häufiger selbsterklärende, gemeinsam
verwendbare und ausführbare Notebooks für die Datananalyse
und maschinelles Lernen. Datalab basiert auf Jupyter
und ist frei verfügbar. So sieht die Datalab-Nutzeroberfläche aus. Sie sehen, dass Chord-Bereiche 
mit Markup und Ausgabe verzahnt sind. Das ist eine Methode, die sich 
als äußerst nützlich bewährt hat. Diese Notebooks eignen sich 
durch eine solche Verzahnung besonders für Datananalyse und ML. Führen Sie den Code durch Klicken auf
"Ausführen" oder Umschalt+Eingabe aus. Die Ausgabe enthält nicht nur
Befehlszeilen, sondern auch Diagramme. Der rote Bereich enthält 
Markup für Erläuterungen und einer der grünen Bereiche
enthält die Option zum Exportieren des Notebooks als eigenständige Datei. Klicken Sie auf 
den anderen grünen Bereich, um Änderungen in einem Code-Repository 
auf Google Cloud Platform zu speichern. In diesem Lab geht es darum, 
wie Sie Daten aus Code-Repositories aufrufen und an sie übergeben.
