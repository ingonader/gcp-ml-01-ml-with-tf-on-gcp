¿Usan cuadernos IPython
o Jupyter en la actualidad? Cada vez más, los científicos de datos
trabajan en cuadernos autodescriptivos ejecutables, que se pueden compartir,
cuando quieren realizar tareas de análisis de datos
o aprendizaje automático. Datalab se basa en Jupyter
y es de código abierto. Esta es la interfaz de Datalab. Observen que hay secciones
de código intercaladas de lenguaje de marcado y salidas. Este intercalado es lo que hace
este estilo de computación tan útil. El análisis de datos
y el aprendizaje automático se realizan comúnmente
en cuadernos como este. Pueden ejecutar el código
si hacen clic en el botón "Run" o si presionan Mayús + Intro. Observen que la salida aquí
no es solo de la línea de comandos también son gráficos. La sección roja contiene
lenguaje de marcado para explicar lo que están haciendo.
Y una de las secciones verdes tiene un botón para exportar el cuaderno
como archivo independiente. La otra sección verde es donde harían clic
si quisieran confirmar los cambios en un repositorio de código
en Google Cloud Platform. En este lab, les mostraremos
cómo solicitar código de repositorios y cómo confirmarlo.
