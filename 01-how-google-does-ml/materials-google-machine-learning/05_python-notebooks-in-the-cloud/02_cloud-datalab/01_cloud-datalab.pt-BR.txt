Você usa blocos de notas do IPython ou do
Jupyter atualmente? Cada vez mais os cientistas de dados
trabalham em blocos de notas autodescritivos, compartilháveis
e executáveis sempre que querem fazer análise de dados ou
aprendizado de máquina. O Datalab se baseia no Jupyter e no 
código aberto dele. A interface do Datalab tem essa aparência. Perceba que há seções de bate-papo em
grupo intercaladas com marcação e saída. Essa intercalação é o que faz o estilo
de computação tão útil. Análise de dados e aprendizado de máquina
são executados comumente em blocos de notas como esse. Você pode executar o código clicando no botão "Executar" ou pressionando
Shift + Enter. Note que a saída aqui não é só uma linha
de comando, são gráficos também. A seção vermelha contém marcação para que
explique o que está fazendo e o motivo. E uma das seções verdes contém um botão para exportar o bloco de notas como
um arquivo independente. A outra sessão verde é onde você clicaria
se quisesse fazer o commit das alterações em um
repositório de código do GCP. Neste laboratório, mostraremos como
extrair e fazer commit para repositórios de código.
