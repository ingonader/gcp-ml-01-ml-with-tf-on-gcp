Voici à quoi ressemble Datalab. C'est un bloc-notes Jupyter
à utiliser où que vous soyez et qui exécute Python, Bash, etc. Par exemple,
une fois dans cette cellule de code, je peux saisir du code Python, "x = 5", "y = x + 5" et "print (y)".
Si je l'exécute, j'obtiens ce résultat. Voilà toute la simplicité
et la puissance du bloc-notes. Le résultat du programme
est incorporé au code même. Je peux ensuite ajouter une note. Je peux ajouter, par exemple,
un titre... Puis du texte... Lors de l'exécution,
j'obtiens une note. Vous avez ainsi du code, des résultats, des commentaires,
le tout dans le même bloc-notes. Lorsque vous utiliserez ces blocs-notes,
il vous arrivera souvent d'effacer toutes les cellules, ce qui entraîne
la suppression des résultats. Vous pouvez aussi
exécuter toutes les cellules, ce qui entraîne de fait leur exécution. Voilà le principe de fonctionnement
du bloc-notes Datalab.
