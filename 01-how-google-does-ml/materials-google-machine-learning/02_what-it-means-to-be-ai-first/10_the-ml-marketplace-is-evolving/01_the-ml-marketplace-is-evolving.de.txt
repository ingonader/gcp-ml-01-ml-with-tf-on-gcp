Der Markt für maschinelles
Lernen verschiebt sich immer mehr in Richtung erhöhter
ML-Abstraktionsebenen. Aucnet entwickelte ein eigenes Modell, um Autoansichten zu klassifizieren
und den Auktionspreis zu schätzen. Dies war ein von Grund
auf neu entwickeltes Modell. Ocado bediente sich einer
API für natürliche Sprachen. Die Ergebnisse
der NL-API wurden analysiert und zur Weiterleitung
von Kunden-E-Mails verwendet. Sie begannen nicht bei
null, sondern eine Ebene höher, mit einer API für natürliche Sprachen. Mit den API-Ergebnissen
zur E-Mail-Art und zu Entitäten konnten E-Mails
leichter weitergeleitet werden. Die dritte Abstraktionsebene
auf dieser Folie ist Giphy. Giphy verwendet die Vision
API zur Textsuche in Memes. Und zwar mithilfe
der optischen Zeichenerkennung. Giphy verwendet also
die Standardversion einer Vision API, um Text in Memes zu erkennen. Ein Unternehmen für soziale
Medien nutzt zudem eine Vision API, um den Upload
anstößiger Inhalte zu verhindern. Auch dieses Beispiel zeigt, wie man die API einsetzen kann,
um ein Geschäftsproblem zu lösen. Das letzte Beispiel ganz rechts
zeigt eine noch höhere Abstraktion. Uniqlo entwickelte einen
Einkaufs-Chatbot mit Dialogflow. Damit konnten sie ein völlig
neues Einkaufserlebnis bieten. So etwas werden Sie immer häufiger sehen. Anstelle der früher
verwendeten benutzerdefinierten Modelle erhöhen wir mit der Zeit
die Ebenen der ML-Abstraktion, wie das oft bei
Softwareprogrammen üblich ist. Zunächst entwickelt man
Dinge auf einer niedrigen Ebene, solange die Technologie
neu und unausgereift ist. Je ausgefeilter die Technologie wird, desto abstrakter und
vielschichtiger werden Prozesse. In diesem Kurs geht es um die Entwicklung
benutzerdefinierter Maschinenmodelle. Wir vermitteln Ihnen
Kenntnisse zum maschinellen Lernen. Die Abstraktionsebene,
auf der wir arbeiten, umfasst die Entwicklung
benutzerdefinierter Modelle. Beachten Sie jedoch: Sie werden maschinelles
Lernen in Zukunft immer häufiger in Form von APIs
in Ihre Anwendungen integrieren. Natürlich muss jemand diese
APIs für den Markt programmieren. Vielleicht werden ja Sie diese APIs
für maschinelles Lernen entwickeln.
