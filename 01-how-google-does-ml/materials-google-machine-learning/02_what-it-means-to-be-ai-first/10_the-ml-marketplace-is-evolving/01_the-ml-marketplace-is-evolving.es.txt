El mercado del AA avanza hacia niveles
cada vez mayores de abstracción de AA. Aucnet creó un modelo para clasificar
piezas de autos y estimar el precio. Pero era un modelo
personalizado creado desde cero. En Ocado lo crearon a partir
de la API de Natural Language (NL). Analizaron los resultados de la API de NL
y los usaron para enrutar los correos. No comenzaron desde cero comenzaron desde un nivel superior ya que tomaron la API de NL
y pudieron usar la opinión y las entidades de esa API como ayuda
para sus decisiones de enrutamiento. Giphy es el tercer
nivel de abstracción en esta diapositiva Usa la API de Vision
para encontrar el texto en los memes y usa el
reconocimiento óptico de caracteres. Aquí, Giphy usa la API de Vision
para encontrar texto en los memes. Una empresa de medios sociales como otro ejemplo, usa una API de Vision
para rechazar las cargas inapropiadas. Ese es otro ejemplo de usar la API
para solucionar un problema de negocios. Y la abstracción final, en el lado derecho es de un nivel mucho más alto. Por ejemplo, Uniqlo diseñó un robot
de chat para compras mediante el uso de Dialogflow y pudieron usarlo para crear una experiencia
de compra completamente nueva. Y esto es algo que verán una y otra vez donde se usaban los modelos
de imagen personalizados. Con el tiempo, iremos
a niveles cada vez más altos de abstracción de AA, siguiendo la ruta
común de la mayoría del software donde se tiende
a crear cosas en un nivel muy bajo cuando la tecnología
es nueva y no se ha desarrollado. Y a medida que madura la tecnología las cosas comienzan a ser más
abstractas y con un nivel mucho más alto. En esta especialización, el enfoque
estará en crear modelos personalizados. Así que, les enseñaremos a usar el AA y el nivel de abstracción
con el que trabajaremos será sobre la creación
de modelos personalizados. Pero tengan presente que,
cada vez más incorporarán el aprendizaje automático
