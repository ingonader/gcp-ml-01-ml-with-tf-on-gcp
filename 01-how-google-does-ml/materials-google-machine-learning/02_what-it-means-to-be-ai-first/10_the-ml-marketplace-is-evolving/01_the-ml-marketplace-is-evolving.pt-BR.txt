O mercado do ML está indo em direção a níveis crescentes
de abstração do aprendizado de máquina. O Aucnet criou
um modelo personalizado próprio para classificar peças de carros
e estimar preços. Mas esse era um modelo personalizado
construído do zero. O Ocado conseguiu se desenvolver
a partir da Natural Language API. Eles analisaram os resultados dessa API e
os usaram para rotear e-mails de clientes. Eles não estavam começando do zero, estavam começando de um nível acima, reiniciando a partir da API,
e eles usaram o sentimento e as entidades dessa API para ajudá-los
nas decisões de roteamento. Giphy, que é o
terceiro nível de abstração neste slide, usa a Vision API para
encontrar texto em memes e usa reconhecimento
óptico de caracteres. Então, o Giphy consegue usar Vision API
para encontrar texto em memes. Uma empresa de mídia social, outro exemplo, usa uma Vision API
para rejeitar uploads inadequados. É outro exemplo de uso da API
para resolver um problema de negócios. E o último, no lado direito, é uma abstração de nível muito maior. Por exemplo, a Uniqlo
projetou um chatbot de compras usando o Dialogflow para criar uma experiência
de compras completamente nova. E isso é algo que você vai ver
cada vez mais, as coisas sendo feitas
com modelos de imagem personalizados. Com o tempo, vamos caminhar
para níveis crescentes de abstração de ML seguindo o caminho
da maioria dos softwares, em que você tende a criar coisas
em um nível muito baixo enquanto a tecnologia
ainda é nova e amadurece. E conforme a tecnologia amadurece, tudo começa a ficar cada vez mais abstrato
e em um nível muito mais alto. Nesta especialização, veremos como criar modelos
de máquina personalizados. Então, vamos ensinar
como desenvolver o aprendizado de máquina, e o nível de abstração
com o qual vamos trabalhar será a criação de modelos personalizados. Mas saiba que, cada vez mais, você conseguirá incorporar
o aprendizado de máquina em seus aplicativos,
principalmente na forma de APIs. Claro, alguém terá que criar essas APIs
para o mercado e, talvez, você seja o criador dessas
APIs de aprendizado de máquina.
