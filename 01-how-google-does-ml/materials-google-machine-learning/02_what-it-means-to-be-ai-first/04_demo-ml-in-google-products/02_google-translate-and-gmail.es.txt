La aplicación Google Traductor les permite
apuntar la cámara del celular a una señal y traduce su significado. Es un buen ejemplo de una combinación
intuitiva de varios modelos. Un modelo para encontrar la señal otro para leerla para realizar el reconocimiento
óptico de caracteres. Un tercer modelo para traducir la señal o tal vez para detectar el lenguaje
y un cuarto modelo para traducir la señal. Un quinto modelo
para superponer el texto traducido. Y hasta un sexto modelo
para seleccionar la fuente. La Respuesta inteligente
es una función de Recibidos en Gmail donde el programa de correo sugiere
tres posibles respuestas a un correo. Esto es lo que veo el modelo de AA en producción
más sofisticado actualmente. ¿Por qué creen que es eso? Es un modelo de secuencia a secuencia es decir, toma un correo como entrada y genera una respuesta como salida. Y el texto, como veremos más adelante se suele considerar
como una secuencia de palabras. Este modelo de aprendizaje automático necesita comprender
un cuerpo de texto pequeño el correo entrante,
y predecir tres respuestas diferentes. Veremos los modelos de secuencias
en el penúltimo curso de la especialización. Tenemos mucho
que abarcar antes de llegar ahí.
