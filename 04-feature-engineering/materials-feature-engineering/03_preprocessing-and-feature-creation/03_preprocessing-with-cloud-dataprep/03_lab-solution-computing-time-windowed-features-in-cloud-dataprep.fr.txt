Dans cet atelier, vous avez appris à mettre en œuvre votre propre tâche Dataflow
sans écrire de code Apache Beam, à l'aide d'une interface utilisateur
graphique simple dans votre navigateur. Vous avez vu que vous pouviez explorer
un ensemble de données complexe incluant des millions de lignes et créer
une caractéristique sophistiquée qui calcule une moyenne mobile avec une bibliothèque prédéfinie de
fonctions de préparation depuis Dataprep. Ce module est terminé. Vous avez beaucoup appris sur la création
de caractéristiques et le prétraitement. J'espère que vous pourrez appliquer ces connaissances
dans le cadre de vos futurs projets. Merci de votre attention.
