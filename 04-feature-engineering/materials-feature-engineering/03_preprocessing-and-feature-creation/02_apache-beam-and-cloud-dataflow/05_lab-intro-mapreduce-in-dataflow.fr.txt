Bienvenue. Vous allez bientôt voir comment, à partir de la mise en œuvre d'un
pipeline simple de l'atelier précédent, ajouter des fonctionnalités de pipeline pour analyser
des instructions d'importation dans des fichiers de code source Java. Le pipeline utilisera des opérations
de type MapReduce pour trouver les cinq packages les plus populaires importés par un exemple
de code source Java. Une fois la mise en œuvre prête, vous exécuterez le pipeline
dans votre environnement Cloud Shell et vérifierez que le résultat correspond
bien aux packages les plus populaires.
