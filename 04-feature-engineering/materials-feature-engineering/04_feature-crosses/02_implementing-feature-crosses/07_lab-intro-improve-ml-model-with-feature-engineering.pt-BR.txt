Neste laboratório,
você aplicará as lições aprendidas neste módulo
para adicionar engenharia de atributos ao modelo de estimador que criamos para
o problema de previsão de tarifas de táxi. Você verá que, ao fazer
a engenharia de atributos, os impactos finalmente começam a
aparecer no RMSC. Você aprenderá a incorporar
a engenharia de atributos no seu canal. Você trabalhará com colunas de atributos, adicionará cruzamentos
de atributos no TensorFlow, e configurará um canal de dados
que leia dados do BigQuery e crie conjuntos de dados
pré-processados usando o Dataflow. Nos dados pré-processados, você treinará
o modelo mais sofisticado.
