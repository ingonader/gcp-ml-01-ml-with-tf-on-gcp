Dans cet atelier, vous allez appliquer
les leçons apprises dans ce module pour ajouter
l'extraction de caractéristiques au modèle d'estimateur de base que nous avons créé pour le problème
de prédiction du prix des courses en taxi. Vous verrez que l'extraction
de caractéristiques commence enfin à jouer sur la RMSC. Vous apprendrez à intégrer
l'extraction de caractéristiques à votre pipeline. Vous travaillerez
avec les colonnes de caractéristiques, ajouterez des croisements
de caractéristiques dans TensorFlow, et configurerez un pipeline de données
qui lit les données de BigQuery et crée des ensembles de données
prétraités avec Dataflow. Vous entraînerez ensuite
votre modèle plus sophistiqué sur les données prétraitées.
