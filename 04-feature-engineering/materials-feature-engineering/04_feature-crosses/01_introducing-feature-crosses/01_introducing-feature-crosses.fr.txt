Bienvenue. Je m'appelle Lak. Je vais vous parler d'une technique d'extraction
de caractéristiques très puissante : les croisements de caractéristiques. Pour le machine learning conventionnel, les croisements de caractéristiques
ne jouent pas un rôle très important, mais, pour les méthodes de ML modernes,
ils constituent un outil précieux. Dans ce module, vous apprendrez
à reconnaître les problèmes pour lesquels
les croisements de caractéristiques facilitent fortement
l'apprentissage des machines. Une fois que vous saurez quand
utiliser cette technique, vous apprendrez
à la mettre en œuvre dans TensorFlow. Vos données brutes ne contiendront pas
de croisements de caractéristiques. Vous devrez les créer à partir
de vos données d'entrée brutes. Vous apprendrez alors à insérer la création de caractéristiques
dans le pipeline de ML. Enfin, vous mettrez en pratique
toutes ces leçons pour améliorer le modèle
sur les tarifs des courses en taxi avec des croisements de caractéristiques.
